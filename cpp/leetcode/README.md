# LeetCode solutions (C++)

Solutions follow **NeetCode 150** order. Each file:

- **Filename**: `NNNN_problem_slug.cpp` (LeetCode problem number + slug).
- **Content**: One problem per file, multiple solutions with short descriptions (algorithm + Time/Space complexity).
- **Naming**: Solution functions like `method1_xxx_xxx()`, `method2_xxx_xxx()`.
- **Main**: One solution is called; others are commented out.

Compile example: `g++ -std=c++17 -o 0001_two_sum 0001_two_sum.cpp`
